Report on design used in website

May 13 2022,

As agsignment suggests, the website consists of five HTML pages:

-home.html

-portfolio.html

-contact.html

-blog.html

-sitereport.html

Home page: While making home page I used light yellow color as it was what I liked and seemed suitable for my web page. The image I used in the home page is my favourite picture. For animation I used keyframes.

Portfolio page: There are two columns where I have used grid-layout.

Contact page: I used border to distinguish contact details. I also have used box-shadow, border-radius, hover to enhance the look of it.

Blog: I used column for this. Column rule for the lines in between.

Report: Heading and paragraphs are used.

Nav bar: Instead of using nav tag i have used div as it was easier for me.

Hamburger icon: I have used Java Script, HTML, CSS to make hamburger responsive.
